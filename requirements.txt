To get the Admit-assist portal running on a new machine, they just need to run these three commands in their terminal:

Create a Virtual Environment:
python -m venv venv

Activate it:

Windows: venv\Scripts\activate

Mac/Linux: source venv/bin/activate

Install everything:
pip install -r requirements.txt
